# Ecommerce-with-R

Det här projektet analyserar e-handelsdata

## Filer

-   `load_data.R` läser in data

-   `script1.R` städar och förbereder data

-   `summary_vis.R` gör analys och skapar grafer och samanfattningar

-   `notebook.R` innehåller hela analysen

### Frågor:

1.  Vilka produktkategorier verkar driva högst försäljning?
2.  Hur skiljer sig ordervärde mellan olika kundsegment?
3.  Försäljning över tid: totalt, regionalt och kundtyp?

## Att köra projektet

Öppna projektet i Rstudio och kör filerna för steg eller kör notebooken

## Paket

-   tidyverse

-   ggplot2
